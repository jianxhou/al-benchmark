# Active Learning Benchmark for Engineering Surrogate Models

A systematic benchmark of acquisition strategies for Bayesian optimization on engineering-flavored problems, with rigorous non-parametric statistical comparison.

## Status

🚧 Project started: May 27, 2026
🎯 Target completion: 5 weeks

## Project Structure
## Environment

- macOS Sonoma 14.4 (Apple Silicon M3)
- Python 3.11
- Conda environment: `oas-test`

## Key Dependencies

- BoTorch / GPyTorch / PyTorch — Bayesian optimization
- OpenAeroStruct — aerospace surrogate problem
- SMT — engineering benchmark functions (Borehole, Piston)
- scikit-posthocs — Friedman / Nemenyi tests
- Polars / pandas — data processing
