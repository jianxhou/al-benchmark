# Scale-disparity failure study (archived)

This folder holds the completed scale-disparity failure study, documented in `docs/lab_notebook.md` (the Phase 2 scale/dimension entry).

The core finding was later found to duplicate Xu & Zhe 2024 (arXiv:2402.02746) and Papenmeier et al. 2025 (arXiv:2502.09198), so it was not pursued for publication.

`exp_scale_failure.py`'s underflow probes (`frac_underflow`, mechanism logging) are retained here as the reference implementation for the acquisition-value underflow instrumentation planned in the reproducibility study.
